The **Pet Adoption Portal** is a full-stack web application built to simplify and digitize the process of adopting pets.  
It connects **users** who wish to adopt with **admins** who manage pet listings, applications, and help requests.

The system enables:
- Real-time pet listings  
- Online adoption applications    
- Dynamic pet management dashboard  
- Help request and response system  

This project demonstrates the use of **Spring Boot**, **Hibernate**, **Thymeleaf**, **MySQL**, and **Bootstrap** in building a complete, responsive, and secure web application.
